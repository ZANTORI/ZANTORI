Política de Privacidad - ZANTORI

Última actualización: Diciembre 2024

1. Información que Recopilamos

ZANTORI no recopila información personal de los usuarios. Nuestra aplicación es una tienda web que funciona completamente offline, utilizando solo archivos HTML locales.

2. Uso de la Información

No almacenamos, procesamos o transmitimos datos personales. La aplicación funciona como un catálogo de productos con funcionalidad de carrito de compras local.

3. Comunicaciones Externas

La aplicación puede abrir WhatsApp para procesar pedidos, pero no recopilamos ni almacenamos información de estas comunicaciones.

4. Seguridad

Implementamos medidas de seguridad estándar de Android para proteger la aplicación y sus datos locales.
